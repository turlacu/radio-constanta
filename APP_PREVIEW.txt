🎨 DESIGN HIGHLIGHTS
══════════════════════════════════════════════════════════════════

✨ VISUAL FEATURES:
   • Deep black gradient background
   • Cyan blue accent color (#00BFFF)
   • Soft rounded corners (16px radius)
   • Subtle card shadows
   • Glowing effects on active elements
   • Smooth fade-in animations

📱 MOBILE OPTIMIZED:
   • Max width: 480px (portrait mode)
   • Large touch targets (44px minimum)
   • No horizontal scroll
   • Smooth scrolling
   • Bottom navigation always accessible

🎭 ANIMATIONS:
   • Fade in on page load
   • Slide up for cards
   • Scale on button press
   • Pulsing live indicator
   • Smooth tab transitions
   • Loading spinner

🎯 USER EXPERIENCE:
   • One-tap play/pause
   • Quick station switching
   • Easy article browsing
   • Full-screen reading
   • Instant navigation
   • Error recovery


══════════════════════════════════════════════════════════════════
⚡ INTERACTIONS
══════════════════════════════════════════════════════════════════

RADIO PAGE:
  Tap Play/Pause   → Start/stop stream
  Tap Station Pill → Switch between FM/Folclor
  Visual feedback  → Button scales on press

NEWS PAGE:
  Tap Article Card → Open full article
  Scroll Down      → See more articles
  Tap Load More    → Fetch older articles
  Tap Back         → Return to feed
  Tap External     → Open in browser

NAVIGATION:
  Tap Radio Tab    → Go to radio player
  Tap Știri Tab    → Go to news feed
  Smooth Animation → Fade between pages

📊 TECHNICAL SPECS
══════════════════════════════════════════════════════════════════

Framework:    React 18 with hooks
Build Tool:   Vite (lightning fast)
Styling:      Tailwind CSS (utility-first)
Animations:   Framer Motion
Routing:      React Router v6
Bundle Size:  ~230KB (gzipped)

Performance:
  First Paint:  < 1 second
  Interactive:  < 2 seconds
  Smooth 60fps: animations
